## 🏗️ Projects & Performance

### 1️⃣ Credit Card Fraud Detection

**Objective:** Identify fraudulent transactions using a balanced classification approach.

* **Model:** Random Forest Classifier
* **Key Achievement:** Achieved an **88% Accuracy** with a strong balance between Precision and Recall.
* **Performance Metrics:**
* **Precision (Fraud):** 0.90
* **Recall (Fraud):** 0.87
* **F1-Score:** 0.88


* **Analysis:** By utilizing undersampling, the model successfully avoids bias towards legitimate transactions, maintaining a high detection rate for fraud.

### 2️⃣ Movie Genre Classification (NLP)

**Objective:** Multi-class classification of movie genres based on IMDb plot summaries.

* **Winning Model:** Logistic Regression (**58.52% Accuracy**)
* **Benchmarking Results:**
* **Logistic Regression:** 58.52%
* **SVM (LinearSVC):** 57.40%
* **SGD Classifier:** 57.08%
* **Multinomial Naive Bayes:** 52.15%


* **Analysis:** In this high-dimensional TF-IDF feature space, Logistic Regression provided the most stable and accurate predictions across the diverse set of movie genres.
